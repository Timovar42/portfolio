# PRD: Одностраничный сайт барбершопа

## Обзор

Одностраничный лендинг для мужского барбершопа. Весь код — в одном файле `index.html` (CSS в `<style>`, JS в `<script>`). Внешние зависимости: только Google Fonts. Без фреймворков, без npm, без бэкенда.

---

## Поведение страницы

### Fullpage scroll-snap

Каждая секция занимает минимум 100vh; при скролле — плавный переход между секциями.

```css
html {
  overflow-y: scroll;
  height: 100%;
  scroll-behavior: smooth;      /* плавные переходы по якорям */
  scroll-padding-top: var(--nav-h);
}
section.snap {
  min-height: 100vh;            /* секции ≈ на весь экран, но без принудительного snap */
  overflow: hidden;
}
```

> Эволюция: `mandatory` → `proximity` → полностью убран scroll-snap. Референс ([yrist-pmr.tilda.ws](https://yrist-pmr.tilda.ws)) использует обычную непрерывную прокрутку — именно scroll-snap создавал «рывки» при свайпе. Теперь скролл нативно плавный.

### Анимация появления (IntersectionObserver)

Элементы с классом `.reveal` появляются при входе в viewport:

```css
.reveal {
  opacity: 0;
  transform: translateY(32px);
  transition: opacity 0.75s ease, transform 0.75s ease;
}
.reveal.visible { opacity: 1; transform: none; }
.d1 { transition-delay: 0.1s; }
.d2 { transition-delay: 0.25s; }
.d3 { transition-delay: 0.4s; }
.d4 { transition-delay: 0.55s; }
.d5 { transition-delay: 0.7s; }
```

- Один инстанс `IntersectionObserver` на все `.reveal`.
- Анимированные счётчики в секции 2 — через `requestAnimationFrame`, запуск при загрузке.

## Цветовая палитра

```css
:root {
  --bg:         #0F0F0F;   /* основной фон — почти чёрный */

  --bg3:        #222222;   /* фон карточек */
  --bg-light:   #F5F2EC;   /* светлая секция (прайс, форма) */
  --copper:     #B5703A;   /* медно-бронзовый акцент */
  --copper-lt:  #D4956A;   /* светлее для hover и текста */
```

- Eyebrow-метки: Inter 11px, uppercase, `letter-spacing: 0.2em`, цвет `--copper-lt`

- `position: fixed`, `top: 0`, `z-index: 1000`
- Фон: `rgba(15,15,15,0.88)`, `backdrop-filter: blur(14px)`
- Высота: 58px

| Элемент | Описание |
|---------|----------|
| Слева | Логотип: «✂» медным + название Bebas Neue 18px |
| Справа | Ссылки: Услуги / Мастера / Контакты (Inter 11px uppercase) + кнопка «Записаться» (`--copper`) |

**Мобильные (< 768px):** ссылки скрыть, только логотип + кнопка «Записаться». Бургер не делать.

## Секции

### 1. HERO (`--bg`)

- Контент по центру (flexbox)
- Фоновая сетка: CSS gradient, ячейка 40px, `rgba(255,255,255,0.03)`
- Eyebrow: «Мужской барбершоп · [Город]»
- H1 Bebas 80px (мобайл 48px → 38px): строка 1 `--cream`, строка 2 `--copper`
- Подзаголовок Inter 15px `--cream40`, max-width 360px
- Кнопки: Primary «Записаться онлайн» / Ghost «Смотреть услуги»
- Scroll-индикатор: «SCROLL» + анимация bounce стрелки (1.8s infinite)

### 2. ПОЧЕМУ МЫ (`--bg2`)

- H2: «НАШИ ПРИНЦИПЫ»
- 4 блока в ряд (мобайл 2×2), вертикальные разделители `--border`
- Блоки: символ/число 36px Bebas `--copper` + название 14px bold + описание 13px `--cream40`
- Счётчик «10+» — анимация при загрузке

### 3. УСЛУГИ (`--bg`)

- H2: «УСЛУГИ», подзаголовок «Всё для безупречного образа»
- Сетка 3 колонки (мобайл 1), 6 карточек
- Карточка: `--bg3`, border `--border`, radius 0, padding 28px 24px
- Медная полоса 2px сверху: width 0→100% при появлении
- Hover: border `--copper`, `translateY(-4px)`

### 4. ПРАЙС (`--bg-light`)

- H2: «СТОИМОСТЬ», подзаголовок про категории мастера
- Меню-стиль: flex, dotted border-bottom, название слева — цена справа
- Кнопка «Записаться» на всю ширину

### 5. МАСТЕРА (`--bg`)

- H2: «НАШИ МАСТЕРА»
- Горизонтальный скролл карточек 200px, скрытый scrollbar
- Заглушки фото: 200×240px, инициалы по центру
- Кнопка ghost «Выбрать мастера и записаться»

### 6. АТМОСФЕРА / ГАЛЕРЕЯ (`--bg2`)

- Две плывущие горизонтальные ленты с фото **причёсок клиентов** (готовые стрижки, а не оборудование/барберы), верхняя движется влево, нижняя — вправо (`reverse`)
- Full-bleed на всю ширину экрана, края растворяются через `mask-image` градиент
- Бесконечная CSS-анимация `translateX`, пауза при наведении, hover убирает grayscale + лёгкий zoom
- Ссылка на Instagram: «Смотреть все работы →»

### 7. ОТЗЫВЫ (`--bg`)

- Рейтинг-бейдж: «★ 4.8 / 5 — на основании Google отзывов»
- 3 карточки отзывов (мобайл 1 колонка)
- Кавычка Bebas 48px `--copper` float left

### 8. ЗАПИСЬ (`--bg-light`)

- Два блока бок о бок (мобайл — колонка)
- Левый: форма (имя, телефон, услуга, мастер) → success-сообщение после submit
- Правый: контакты + кнопка Telegram
- Примечание про Altegio

### 9. ФУТЕР (`--bg`)

- Высота auto, не 100vh
- 3 колонки: логотип+описание / навигация / соцсети
- Соцсети: кружки 36px с буквами ig / tg / fb / vb / tt
- Копирайт

## Мобильная адаптация (< 768px)

- scroll-snap сохраняется, `min-height: 100vh`
- H1 hero: 38px
- 3-колонные и 2-колонные сетки → 1 колонка
- Горизонтальный скролл мастеров без изменений
- Форма: одна колонка, форма сверху
- Навигация: логотип + кнопка
