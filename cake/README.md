# Сладкие Торты — сайт домашнего кондитера

Статическая версия для портфолио: bento-лендинг, конструктор тортов, каталог готовых позиций.

## Сборка

```bash
npm install
npm run build
```

Готовый сайт появится в папке `out/`.

**Как открыть:** двойной щелчок по `out/index.html` или по `Открыть сайт.bat` в корне проекта.

> Не открывайте папку `out/` в браузере — покажется список файлов, а не сайт.

> После изменений в коде пересобирайте: `npm run build`

## Разработка (опционально)

```bash
npm run dev
```

Сайт откроется на http://localhost:3000.

## Структура

- `src/lib/pricing/calculatePrice.ts` — расчёт цены
- `src/lib/data/catalog.ts` — контент сайта (настройки, каталог, портфолио, опции конструктора)
- `out/` — собранный статический сайт (после `npm run build`)

## Страницы

| Путь | Описание |
|------|----------|
| `/` | Портфолио с фильтрами |
| `/constructor` | Пошаговый конструктор |
| `/catalog` | Готовые торты |
| `/catalog/[id]/order` | Заказ готового торта |

Заявки сохраняются локально: текст копируется в буфер обмена, клиент звонит кондитеру.
