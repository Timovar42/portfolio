# PRD: Сайт-портфолио BarbieBoys

## Что делаем
Одностраничный сайт-портфолио веб-агентства BarbieBoys.
Цель — потенциальный клиент, получив ссылку, за 2-3 минуты
понимает что делает агентство, видит реальные работы
и хочет написать.

## Кто пользователь
Владелец малого бизнеса (барбершоп, кондитерская, юрист,
фитнес-тренер и т.п.) которому прислали ссылку на портфолио.
Он не разбирается в технологиях — ему важно:
«красиво ли?», «делали ли что-то похожее на моё?»,
«как с ними связаться?»

## Дизайн-система

**Концепция:** тёмный, современный, уверенный.
Не корпоративный и не игривый — агентство которое
делает дело и знает себе цену. Ближайшие ориентиры:
сайты Linear, Vercel, маленьких европейских студий.

```css
:root {
  /* Основная палитра */
  --bg:         #0C0C0F;   /* почти чёрный фон */
  --bg2:        #131318;   /* чуть светлее — секции */
  --bg-card:    #1A1A22;   /* фон карточек */
  --accent:     #6C63FF;   /* фиолетовый акцент — кнопки */
  --accent2:    #9B94FF;   /* светлее — hover, текст на тёмном */
  --white:      #FFFFFF;
  --white80:    rgba(255,255,255,0.80);
  --white40:    rgba(255,255,255,0.40);
  --white08:    rgba(255,255,255,0.08);
  --border:     rgba(255,255,255,0.07);

  /* Типографика */
  --font-display: 'Space Grotesk', system-ui, sans-serif;
  --font-body:    'Inter', system-ui, sans-serif;
}
```

**Почему фиолетовый акцент, а не золото/медь:**
Золото — это про юридические фирмы и барбершопы (уже
использовано в портфолио). Медь — про мастерство рук.
Фиолетовый / индиго — технологии, ИИ, цифровые агентства.
Это сразу читается как «айти», а не как один из клиентов.

**Шрифты (Google Fonts):**
```html
<link href="https://fonts.googleapis.com/css2?
  family=Space+Grotesk:wght@400;500;600;700
  &family=Inter:wght@400;500
  &display=swap" rel="stylesheet">
```
Space Grotesk — современный, технологичный, с характером.
Достаточно необычный чтобы запомниться, достаточно читаемый
чтобы не мешать.

**Карточки проектов — подача через мокап устройства:**
Каждый проект показывается как скриншот в рамке браузера
или телефона (CSS-only мокап, без картинок). Это выглядит
профессионально и не требует внешних ресурсов.

**Signature-элемент:**
Градиентная линия-акцент — тонкая (2px) горизонтальная
полоса под Hero, переходящая из #6C63FF в #FF6584 (розовый).
Этот же градиент появляется как border-top на карточках
проектов при hover. Единственное цветное «украшение» на сайте
— всё остальное монохромно.

## Технический стек

- **Всё в одном файле:** `index.html` (CSS в `<style>`,
  JS в `<script>`) — легко деплоить куда угодно
- **Форма контакта:** отправка в Telegram через
  Telegram Bot API (webhook) — без бэкенда, прямо из JS.
  Формат: `fetch('https://api.telegram.org/bot{TOKEN}/sendMessage')`
- **Деплой:** Netlify или GitHub Pages — бесплатно,
  работает из одного HTML файла
- **Мокапы проектов:** CSS-only рамки браузера/телефона
  с `<iframe>` или скриншотами внутри
- **Анимации:** IntersectionObserver + CSS transitions.
  Никаких библиотек.

## Что НЕ делаем

- Не делаем CMS или админку — контент статичный
- Не подключаем платёжные системы
- Не делаем мультиязычность на первой версии
- Не используем React/Vue/Next.js —
  портфолио должно открываться мгновенно
  без JS-фреймворков
